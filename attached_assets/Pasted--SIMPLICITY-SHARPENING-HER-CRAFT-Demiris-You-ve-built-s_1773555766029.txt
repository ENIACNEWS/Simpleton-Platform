toolRegistry.registerTool({
  name: 'appraise_with_history',
  description: 'Appraise using both spot prices and actual transaction data',
  handler: async (args) => {
    const spotPrice = await getSpotPrice(args);
    const transactionAvg = await getAvgTransactionPrice({
      ...args,
      days: 90,
      region: args.region
    });
    
    // Weighted average: 70% spot, 30% transaction data initially
    // Weight shifts toward transactions as sample size grows
    const sampleSize = transactionAvg.count;
    const spotWeight = Math.max(0.5, 1 - (sampleSize / 100));
    const transactionWeight = 1 - spotWeight;
    
    const blendedPrice = (spotPrice * spotWeight) + (transactionAvg.price * transactionWeight);
    
    return {
      appraisedValue: blendedPrice,
      confidence: sampleSize > 50 ? 'high' : 'medium',
      source: sampleSize > 0 ? 'Spot + Transaction History' : 'Spot Only',
      transactionCount: sampleSize
    };
  }
});
```

**The upgrade:** Every sale makes her smarter. She becomes the most accurate appraiser in your network.

// Email intake
async function processIncomingEmail(rawEmail) {
  // 1. Classify
  const category = await simplicity.classifyEmail(rawEmail);
  // categories: inquiry, offer, invoice, spam, personal, etc.
  
  // 2. Extract key data
  const extracted = await simplicity.extractEntities(rawEmail);
  // entities: prices, dates, items, names, amounts
  
  // 3. Take action based on category
  switch(category) {
    case 'inquiry':
      // Generate draft response with relevant market data
      const draft = await simplicity.generateQuoteResponse(extracted);
      await saveToDrafts(draft);
      break;
      
    case 'offer':
      // Compare to market, flag if good deal
      const analysis = await simplicity.analyzeOffer(extracted);
      if (analysis.isGoodDeal) {
        await notifyUser(`Good deal detected: ${analysis.summary}`);
      }
      break;
      
    case 'invoice':
      // Extract amounts, match to transactions
      await matchInvoiceToTransaction(extracted);
      await updateAccounting(extracted);
      break;
      
    case 'spam':
      await markAsSpam(rawEmail);
      break;
  }
  
  // 4. Store for learning
  await memory.store({
    type: 'email',
    category,
    extracted,
    timestamp: new Date()
  });
}
```

**The upgrade:** Your inbox becomes a **data feed** and a **decision engine**, not a chore.
